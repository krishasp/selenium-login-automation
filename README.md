# Selenium Login Automation

## Project Overview
This project automates login functionality testing using Selenium WebDriver and Mocha.

##Tech Stack
- JavaScript
- Selenium WebDriver
- Mocha

##Test Scenarios
- successful login with valid credentials
- Failed login with invalid password

## Project Structure
- pages/ → Page Object Model
- tests/ → Test cases

## How to Run
npm install
npx mocha tests/loginTest.js

##Output
Both test cases pass successfully ensuring login functionality works as expected.
